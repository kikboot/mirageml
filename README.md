# MirageML - Визуальный HTML/CSS редактор

MirageML - это интуитивно понятный визуальный редактор для создания HTML/CSS разметки с помощью интерфейса drag-and-drop. Проект позволяет быстро прототипировать веб-интерфейсы без написания кода вручную.

Данный сайт разрабатывает 1 человек
 
## Основные возможности

- Визуальное редактирование элементов через интерфейс нажатия на нужный элемент
- Библиотека компонентов: кнопки, блоки, текст, изображения, фигуры
- Редактор свойств: изменение размеров, цветов, отступов в реальном времени
- Система слоев для управления элементами
- Предпросмотр созданного интерфейса
- Помощь с функционалом сайта
- Экспорт в чистый HTML/CSS или ZIP-архив
- Загрузка изображений** прямо в проект

## Быстрый старт

## Требования
- Node.js 16+
- Современный браузер (Chrome, Firefox, Edge)

## Установка
1. Клонируйте репозиторий:
- git clone https://gitlab.com/kikboot/mirageml.git
- cd mirageml

2. Пройти в backend-часть сайта
- cd mirageml-backend

3. Запустить сервер
- node server.js

4. Ссылка
- после выполнения 3 пункта появится ссылка

## Использование
- Добавление элементов: кликните по элементу для автоматического добавления в центр

## Редактирование: 
- Двойной клик по текстовым элементам для изменения содержимого
- Перетаскивание элементов по холсту
- Изменение размеров с помощью маркеров
- Настройка свойств в правой панели

## Предпросмотр:
- Нажмите на предпросмотр, чтобы посмотреть, как будет выглядить страница

## Помощь:
- Нажмите на помощь, чтобы ознакомиться с функционалом сайта

## Экспорт: 
- Нажмите кнопку "Экспорт" в верхней панели
- Выберите нужный формат (HTML, CSS или ZIP)

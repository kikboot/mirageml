Начало работы

Первые шаги с MirageML

Создание первого проекта

После регистрации в системе вы можете создать свой первый проект:

  1. Нажмите кнопку "Создать проект" в личном кабинете
  2. Введите название проекта
  3. Выберите размер холста (по умолчанию 1024×768)
  4. Начните добавлять элементы на холст

Интерфейс редактора

Интерфейс редактора состоит из нескольких основных частей:

  • Панель элементов - слева, содержит все доступные элементы
  • Рабочая область - центральная часть, где создается проект
  • Панель свойств - справа, позволяет настраивать выбранный элемент
  • Панель слоев - вкладка справа, позволяет управлять элементами

Руководство по редактору

Подробное описание всех функций

Добавление элементов

Существует два способа добавления элементов на холст:

  • Клик по элементу - элемент добавляется в центр холста
  • Перетаскивание - элемент можно разместить в нужном месте

Редактирование элементов

Для редактирования элемента:

  • Кликните по элементу для выделения
  • Используйте панель свойств для изменения параметров
  • Перетаскивайте элемент для изменения позиции
  • Используйте маркеры для изменения размера

Работа с текстом

Для редактирования текста дважды кликните по текстовому элементу. Вы можете изменить содержимое, шрифт, размер и цвет.

Экспорт проектов

Как экспортировать и использовать проекты

Форматы экспорта

Вы можете экспортировать проект в следующих форматах:

  • HTML код - готовый HTML-файл для веб-сайта
  • CSS стили - отдельный файл со стилями
  • ZIP архив - полный проект с изображениями и файлами

Процесс экспорта

  1. Завершите создание проекта в редакторе
  2. Нажмите кнопку "Экспорт" в верхней панели
  3. Выберите нужный формат экспорта
  4. Скачайте файлы на ваш компьютер
  5. Используйте файлы на вашем сайте или передайте разработчику

Устранение неполадок

Решение распространенных проблем

Часто встречающиеся проблемы

Элементы не сохраняются

Убедитесь, что вы нажали кнопку "Сохранить" в редакторе. Проекты также автоматически сохраняются при закрытии вкладки.

Экспорт не работает

Проверьте, подключен ли интернет и не блокирует ли ваш браузер всплывающие окна. Некоторые браузеры могут блокировать скачивание файлов.

Медленная работа редактора

Попробуйте обновить страницу или использовать более производительное устройство. Большое количество элементов может замедлять работу.

Частые вопросы

Ответы на популярные вопросы

Могу ли я использовать MirageML бесплатно?

Да, MirageML предлагает бесплатный тариф с базовыми возможностями. Для дополнительных функций и проектов доступны платные тарифы.

Нужно ли знать HTML/CSS для использования редактора?

Нет, MirageML создан для пользователей без навыков программирования. Всё делается визуально через интерфейс.

Могу ли я использовать созданные проекты на своем сайте?

Да, вы можете экспортировать проекты в формате HTML/CSS и использовать их на любом сайте или передать разработчику для интеграции.

Как восстановить доступ к аккаунту?

Используйте функцию "Забыли пароль" на странице входа. Ссылка для восстановления будет отправлена на вашу электронную почту.
